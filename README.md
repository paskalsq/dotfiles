# dotfiles

Personal dotfiles for my Linux setup.

Includes configuration for:

- Hyprland
- Zsh
- Neovim (NvChad)
- Waybar, Rofi, Dunst, etc.
- GTK and QT theming
- install.sh for automated setup

## Usage

Clone the repo:

```bash
git clone https://github.com/paskalsq/dotfiles.git
cd dotfiles
./install.sh
